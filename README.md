C2 任务收尾——萌芽镇 4 栋建筑里的 3 栋（阿降小屋已在 C-1）现在都能进了。

## 这一波加了什么

### 2 个新室内地图

**典籍阁** (`librarian-library.json`)
- 满墙书架（用了新的高书架 tile，4 层、底下放卷轴）
- 中央绿色学者地毯 + 阅读桌
- 蓁站在桌前
- 上下左右都有书架——**视觉上就是图书馆该有的样子**

**铁匠铺** (`blacksmith-forge.json`)
- 灰色石头地板（不是木地板了，氛围不同）
- 上方一对炉子（橙色火焰可见 + 烟囱）
- 中央铁砧（旁边靠着一把锤子）
- 两个工作台
- 角落有个手册书架（铁匠也读书）
- 老周站在铁砧旁，朝右

### 5 块新 tile（加在原 tileset 第 3 行）

| ID | tile |
|---|---|
| 11 | 石头地板 |
| 12 | 炉子（带火焰） |

### 通用化的"门口检测"

之前 C-1 的代码是写死给阿降小屋一个门用。现在重构成**门口列表**——加新门只需要在数组里追加一项。

未来共创之都 8 个工坊，每个工坊就是 `+1 行 InteriorConfig + 1 行 doors[]` ——没了。

### 外景调整

3 个外景 NPC 的位置都挪了一格给门口让位：
- 阿降：(13, 5) → (13, 6)（C-1 已挪）
- 蓁：(21, 5) → (21, 6)
- 老周：(5, 14) → (5, 15)

视觉上几乎看不出来，但门口检测和 NPC 互动彻底分开。

### 测试路线

1. 进游戏 → 进入萌芽镇
2. 走到阿降身边正后方（屋下方门口） → `[E] 进入阿降的小屋` → 按 E → 进屋 → 出屋
3. 走到典籍阁（地图右上的小屋）下方 → `[E] 进入典籍阁` → 进屋
4. **看典籍阁内**：满墙书架 + 绿地毯 + 蓁在桌前 → 跟蓁对话
5. 出屋 → 走到铁匠铺（地图左下）下方 → `[E] 进入铁匠铺` → 进屋
6. **看铁匠铺内**：石头地板 + 火焰熊熊的炉子 + 铁砧 + 老周 → 跟老周对话
7. 出来再绕一圈，确认 3 个门口都能正常开关

### 你应该看到

- ✅ 3 个门口都有 `[E] 进入XXX` 红色提示
- ✅ 进每栋楼过场流畅（300ms 黑屏）
- ✅ 典籍阁感觉像图书馆（满满的书）
- ✅ 铁匠铺感觉像铁匠铺（石地、火焰、铁砧）
- ✅ 出门后回到原位置（不会传送回出生点）
- ✅ 跨场景的任务进度保留（在屋外完成 5/9，进屋出屋还是 5/9）
