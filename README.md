## 4 个任务的具体玩法

### 铁匠任务（最有"游戏感"）

1. 拿徽章后跟铁匠对话 → 他让你"找矿石"
2. **地图东南角水池南边的沙地上，会出现一块带金色矿脉的石头**（轻微淡入淡出动画）
3. 走过去按 E 捡起
4. 回到铁匠按 E → 他给你「铁匠的助手」称号

### 商人任务（"NPC 之间跑腿"经典）

1. 跟商人对话 → 他让你"问问村长今天市集开不开"
2. 走到阿降那里按 E → 阿降会有一段**专门的对话**（"开啊，开了一辈子"）
3. 回到商人按 E → 他给你「可靠的信使」称号

### 图书管理员任务（最低强度）

1. 跟蓁对话 → 她让你"翻这本书到第 3 页"
2. 跟她**第 1 次对话**：你翻开第一页（小镇地图）
3. 跟她**第 2 次对话**：第二页（一首没写完的诗）
4. 跟她**第 3 次对话**：第三页空白 → 称号到手

### 钓鱼老人任务（最考验耐心）

1. 跟默对话 → 他让你"在这里坐 15 秒"
2. **真的不动 15 秒**，期间再跟他对话会显示"还差 X 秒"
3. **走太远（80 像素以外）计时会重置**——别中途跑去摸鱼
4. 15 秒到，再跟他对话拿到「耐心者」称号

## 已知小问题

1. **钓鱼老人任务"走太远重置"** —— 是故意的设计（强迫真的"等"），但有些玩家可能觉得苛刻。如果反馈强烈可以放宽
2. **图书管理员的 3 次互动** —— 翻书的"仪式"很轻，玩家可能没意识到这就是任务。如果反馈说"我不知道做完没"可以加进度提示
3. **矿石的位置在水池南边沙地** —— 地图东南角。如果玩家迷路，可以提示位置
